### Module III

|    **Tools**     |          **Input**          |                          **Output**                          |          Applications           |
| :--------------: | :-------------------------: | :----------------------------------------------------------: | :-----------------------------: |

The table above briefly describes each function in this module. The sample data are displayed through a hyperlink. Users can preview the results by clicking it.
